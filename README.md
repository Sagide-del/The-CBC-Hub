CBC Hub - Competency Based Curriculum Resources

📚 Overview
CBC Hub is a comprehensive educational platform providing quality resources for Kenyan students, teachers, and inclusive learning following the Competency-Based Curriculum.

 Key Features

### For Students
- Resources from PP1 to Form 4
- Notes, worksheets, and past papers
- Pay-per-download model (KSh 30-70)
- Bloom's Taxonomy integrated content

### For Teachers
- Lesson plans and schemes of work
- Teaching notes and assessments
- Professional development resources

### Inclusive Learning
- Resources for 12+ learning conditions
- Visual impairment, dyslexia, autism, etc.
- Adapted and specialized content

### Marketplace
- **School Section**: Bulk supplies (robotics, beads, art)
- **Home DIY**: Individual kits for families
- M-Pesa payments (Till: 345670)
- Availability checker

### Admin Dashboard
- Resource management
- Finance tracking
- Customer requests
- Analytics and reports

## 🛠️ Tech Stack
- **Frontend**: Next.js 16.1.6
- **CMS**: Sanity.io
- **Styling**: Tailwind CSS
- **Payments**: M-Pesa Daraja API
- **Hosting**: Vercel
